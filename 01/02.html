1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta ul sirve para crear una crea una lista no ordenada.
Esta lista la podriamos utilizar para las cosas que tenemos que comprar en el supermercado.

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

El elemento ol permite definir listas o viñetas ordenadas, bien con numeración o alfabéticamente.
Esto lo podriamos utilizar por ejemplo cuando tenemos que copiar los pasos para una receta

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

La etiqueta dl representa una lista descriptiva. El elemento encierra una lista de grupos de términos y de descripciones. Algunos usos comunes para este elemento son implementar un glosario o para desplegar metadatos.

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

El elemento li declara cada uno de los elementos de una lista.
Es decir,despues de poner cualquier otro tipo de lista antes podriamos la etiqueta li
